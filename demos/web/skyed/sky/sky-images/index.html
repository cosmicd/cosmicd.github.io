This is a higher level interface for exploring sky images. To search images, you need to know either the name OR id OR right_ascension(RA)-and-declanation(DEC) of an object in the sky. For beginners, an easy way to get the names or ids of popular objects is the Messier objects list. As an example, search for one of the following: andromeda, M81, NGC1097. Or use RA and DEC (both in degrees): 254.3 -4.1. Another interesting class of objects is galaxy interactions and collisions. For example, try to search for one of the following: NGC2207, NGC1097, NGC4676, NGC4038, NGC520, NGC5194.

If you do not see the object, you may need to change the zoom or field of view (fov) in degrees; fov field provides a quick way to change zoom level.

The default survey is "P/DSS2/color" . This survey does not include solar system objects. The survey list has the available surveys (look for the heading "HiPS images" and column name "ID", and make sure aladinlite supports the survey by looking at column "aladin client"). For example, try this survey: P/allWISE/color.
